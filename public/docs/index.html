Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Admin endpoints

POST Travel Creates a new Travel record.

requires authentication

Example request:
curl --request POST \
    "http://localhost/api/v1/admin/travels" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"is_public\": false,
    \"name\": \"iusto\",
    \"description\": \"repellat\",
    \"number_of_days\": 17
}"

const url = new URL(
    "http://localhost/api/v1/admin/travels"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "is_public": false,
    "name": "iusto",
    "description": "repellat",
    "number_of_days": 17
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{"data":{"id":"996a36ca-2693-4901-9c55-7136e68d81d5","name":"My new travel 234","slug":"my-new-travel-234", ...}

Example response (422):


{
    "message": "The name has already been taken.",
    "errors": {
        "name": [
            "The name has already been taken."
        ]
    }
}

Public endpoints

GET Travels Returns paginated list of travels.

Example request:
curl --request GET \
    --get "http://localhost/api/v1/travels?page=1" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/v1/travels"
);

const params = {
    "page": "1",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Example response (200):


{"data":[{"id":"9958e389-5edf-48eb-8ecd-e058985cf3ce","name":"First travel", ...}}

GET Travel Tours Returns paginated list of tours by travel slug.

Example request:
curl --request GET \
    --get "http://localhost/api/v1/travels/&quot;first-travel&quot;/tours" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"priceFrom\": \"\\\"123.45\\\"\",
    \"priceTo\": \"\\\"234.56\\\"\",
    \"dateFrom\": \"\\\"2023-06-01\\\"\",
    \"dateTo\": \"\\\"2023-07-01\\\"\",
    \"sortBy\": \"\\\"price\\\"\",
    \"sortOrder\": \"\\\"asc\\\" or \\\"desc\\\"\"
}"

const url = new URL(
    "http://localhost/api/v1/travels/&quot;first-travel&quot;/tours"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "priceFrom": "\"123.45\"",
    "priceTo": "\"234.56\"",
    "dateFrom": "\"2023-06-01\"",
    "dateTo": "\"2023-07-01\"",

fetch(url, {
